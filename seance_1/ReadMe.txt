--How to install R for Visual Studio Code--

1- R in Visual Studio Code : https://code.visualstudio.com/docs/languages/r

2- Download R base : https://cloud.r-project.org/

3- Download RStudio : https://posit.co/download/rstudio-desktop/

4- Install vscode-pandoc extension to preview rmd  : https://marketplace.visualstudio.com/items?itemName=ChrisChinchilla.vscode-pandoc

5- (Optional) if the rendering of pandoc is not working, install pandoc by following this link : https://github.com/REditorSupport/vscode-R/wiki/R-Markdown

6- Install live preview extension to preview html file :  https://marketplace.visualstudio.com/items?itemName=ms-vscode.live-server
